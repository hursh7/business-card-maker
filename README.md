### About the Project  
* ```React``` 의 **HOOKS** 방식을 사용하여 개발한 웹 어플리케이션.  
* ```React Router``` 학습.  
* ```Firebase``` 의 **Auth** API를 통해 소셜 미디어 로그인을 구현하고, **Real Time database**를 이용해 사용자 정보를 실시간으로 연동하는 명함 앱.  
* ```Cloudinary``` 로 이미지 업로드 기능 구현.
